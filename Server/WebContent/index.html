AUTOCARTAS APIREST

Listado de Endpoints y descripción corta
Endpoint Descripción
/login Busca el usuario en la base de datos y lo asigna a una variable local.
- POST = http://localhost:8081/AUTOCARTASREST/rest/restcoches/login
/logout Busca la sesión y en caso de encontrarla la borra y asigna el usuario del API a null.
- POST = http://localhost:8081/AUTOCARTASREST/rest/restcoches/logout
/newgame Comprueba que la sesion no tenga una partida y la crea, en caso de haber una la devuelve.
- POST = http://localhost:8081/AUTOCARTASREST/rest/restcoches/newgame
/resetgame Busca y elimina la partida de ese usuario y le crea una nueva.
- POST = http://localhost:8081/AUTOCARTASREST/rest/restcoches/resetgame
/raffle Elige un usuario de forma aleatoria indicando cual va a ser el primero en jugar.
- POST = http://localhost:8081/AUTOCARTASREST/rest/restcoches/raffle
/playcard recibe las elecciones del usuario y ejecuta la partida.
- POST = http://localhost:8081/AUTOCARTASREST/rest/restcoches/playcard
/ready Asigna las variables necesarias para el CPU y avisa al usuario para que haga su jugada.
- POST = http://localhost:8081/AUTOCARTASREST/rest/restcoches/ready
/stats Devuelve un listado de usuarios ordenados en base a sus estadísticas.
- POST = http://localhost:8081/AUTOCARTASREST/rest/restcoches/stats
CRUD
/users Devuelve un listado de los usuarios registrados en la base de datos.
- GET = http://localhost:8081/AUTOCARTASREST/rest/restcoches/raffle
/users/{user} Busca el usuario indicado y en caso de encontrarlo lo devuelve.
- GET = http://localhost:8081/AUTOCARTASREST/rest/restcoches/users/{user}
/users Crea un nuevo usuario en la base de datos y avisa del resultado de la operación.
- POST = http://localhost:8081/AUTOCARTASREST/rest/restcoches/users
/users/{user}/{attr}/{newvalue} Edita el campo del usuario indicado por el valor nuevo también indicado.
- PUT = http://localhost:8081/AUTOCARTASREST/rest/restcoches/users/{user}/{attr}/{newvalue}
/users/{user} Intenta eliminar el usuario indicado y avisa del resultado.
- DELETE = http://localhost:8081/AUTOCARTASREST/rest/restcoches/users/{user}
/coches Devuelve un listado de los coches registrados en la base de datos.
- GET = http://localhost:8081/AUTOCARTASREST/rest/restcoches/coches
/coches/{id} Busca el coche con el id indicado y en caso de encontrarlo lo devuelve.
- GET = http://localhost:8081/AUTOCARTASREST/rest/restcoches/coches/{id}
/coches Crea un nuevo coche en la base de datos y avisa del resultado de la operación.
- POST = http://localhost:8081/AUTOCARTASREST/rest/restcoches/coches
/coches/{idCoche}/{attr}/{newvalue} Edita el campo del coche indicado por el valor nuevo también indicado.
- PUT = http://localhost:8081/AUTOCARTASREST/rest/restcoches/coches/{idCoche}/{attr}/{newvalue}
/coches/{idCoche} Intenta eliminar el coche con el id indicado y avisa del resultado.
- DELETE = http://localhost:8081/AUTOCARTASREST/rest/restcoches/coches/{idCoche}
